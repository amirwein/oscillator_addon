# oscillator_addon
FEL (Free Electron Laser) oscillator add on for the GINGER3D Package

## Contents
- 'cent/': Main files.
- 'cent/ginger/: Files for interaction with the GINGER3D.
- 'cnet/tst/': files for testing a first run.

## Requirements
- MATLAB R2023a+
- GINGER3D
